## 🎯 О проекте

Это моё **личное портфолио-сайт**, где я собираю проекты, которыми горжусь. Здесь сочетаются:
- **Дизайн** — Figma, Photoshop, иллюстрация с планшета
- **Разработка** — Python, JavaScript, HTML/CSS
- **Фотография** — как отдельное направление и источник вдохновения

Сайт выполнен в тёмной эстетике с акцентными цветами `#7F77DD` и `#4ecba0`, использует **Parallax Scrolling**, анимированные частицы и горизонтальный скролл проектов.
